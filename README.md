# 🚗 Car Rental Management System

## 📖 Overview
The Car Rental Management System is a desktop application developed using Java and Java Swing as part of an Object-Oriented Programming (OOP) mini project. It provides a simple graphical interface to manage car rentals efficiently.

## ✨ Features
- Add new cars
- View available cars
- Rent a car
- Return a rented car
- Customer management
- User-friendly GUI
- Object-Oriented design

## 🛠️ Technologies Used
- Java
- Java Swing
- Object-Oriented Programming (OOP)

## 📂 Project Structure
```
CarRentalSystem/
│── src/
│   ├── Main.java
│   ├── Car.java
│   ├── Customer.java
│   ├── Rental.java
│   └── GUI.java
│── images/

```

## 🚀 How to Run
1. Clone this repository.
2. Open it in Eclipse, IntelliJ IDEA, or NetBeans.
3. Compile the project.
4. Run `Main.java`.

## 🎯 Learning Outcomes
- Applied Object-Oriented Programming principles
- Built a desktop GUI using Java Swing
- Implemented classes, objects, inheritance, encapsulation, and polymorphism
- Improved Java programming and GUI development skills
